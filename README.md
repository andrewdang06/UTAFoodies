# MavEats

MavEats is a simple web app that helps UTA students find food near campus. The goal is to make it easy to see nearby restaurants that are actually open, relevant to what you're searching for, and close enough to grab during a study break.

The app focuses on practical searches like halal food, late-night spots, cheap eats, boba, and other places students commonly look for around Arlington.

## Features

* Search for food by category or keyword
* Shows restaurants near UTA
* Prioritizes places that are open and relevant to the search
* Filters out permanently closed locations
* Simple interface meant for quick decisions when you're hungry

## Why this project exists

I want a website specific to UTA students where they can just prompt what they want to eat and they can see all the nearest places without having to scourage through Google Maps.

## Tech Stack

* Frontend: JavaScript / Web UI
* Backend: API-based search logic
* Location data: restaurant listings around UTA

## Status

This project is still being improved. Future updates will focus on better search accuracy, more restaurants near campus, and better late-night filtering.
